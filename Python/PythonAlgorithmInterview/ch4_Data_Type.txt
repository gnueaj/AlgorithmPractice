ㆍ파이썬 자료형
- None (class None Type)

- 숫자
 1) 정수형으로 int만을 제공하며 Arbitrary-Precision을 사용, 정수를 숫자의 배열로 간주하여 오버플로우 방지
 2) bool은 논리 자료형이지만 내부적으로 1과 0으로 처리되는 int의 서브클래스임

- Mapping: 키와 자료형으로 구성된 복합 자료형이며 파이썬에 내장된 유일한 매핑 자료형은 딕셔너리임

- 집합
 1) 중복된 값을 갖지 않는 자료형이며, 빈 집합은 a = set()과 같은 형태로 선언함
 2) 빈 집합이 아닌 집합을 선언할 때는 a = {1, 2, 3}과 같은 형태로 쓰며, 딕셔너리와 같이 중괄호를 사용
 3) set은 입력 순서가 유지되지 않으며, 중복된 값이 있을 경우 하나의 값만 유지됨

- 시퀀스
 1) str, list와 같이 어떤 특정 대상의 순서 있는 나열을 뜻함
 2) 파이썬에서는 list라는 시퀀스 타입이 사실상 배열의 역할을 수행함
 3) 불변 자료형에는 str, tuple, bytes가 있으며, 가변 자료형에는 list가 있음

- 파이썬은 원시 타입을 제공하지 않으며, 객체를 지원함
 1) 불변 객체에는 bool, int, float, tuple, str이 있으며, 가변 객체에는 list, set, dict가 있음
 2) 파이썬은 모든 것이 객체이며, 값을 담고 있는 변수는 사실 참조일 뿐임
  ex) >>> 10
      >>> a = 10
      >>> b = a
      >>> id(10), id(a), id(b) # a가 11이 된다고 해서 b의 값이 변하지 않음
      2130338316816 2130338316816 2130338316816 
 3) 가변 시퀀스 중 하나로 리스트가 있으며, 다른 변수가 참조하고 있을 때 그 변수의 값 또한 변경됨

- is는 id()값을 비교하는 함수이며, None은 null로 값 자체가 정의되어 있지 않으므로 is로만 비교가능
- ==는 값을 비교하는 연산자이며 deepcopy로 복사한 경우 값은 같지만 id는 다르므로 ==는 True, is는 False가 됨

- 파이썬의 객체 구조는 편리하고 강력한 기능을 제공하지만 느린 이유 중 하나임
