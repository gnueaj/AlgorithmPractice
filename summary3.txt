6. 반복문
ㆍwhile문
while count <= 5:
    count += 1;
    if count == 2 :
        continue
    if count == 3 :
        break
else:
    print('No') # break문이 호출되지 않은 경우

ㆍrange() 
- range((start,)stop(,step))
- 숫자 시퀀스를 생성하며, 순회 가능한 객체를 반환함

ㆍfor문
for x in range(0,3):
    print(x)

word = 'thud'
for letter in word:
    print(letter) # break, continue, else는 똑같이 동작함
